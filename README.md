# FrameworkWeb – Projets & Travaux Pratiques

Ce dépôt regroupe l'ensemble des Travaux Pratiques (TP1 à TP6) ainsi qu'un Contrôle Terminal (CT) réalisés dans le cadre de la Licence 3 Informatique à l'Université d'Orléans, pour l'UE Framework Web (semestre 1).

## Présentation

Chaque dossier correspond à un TP ou au CT, avec pour objectif la mise en pratique progressive des concepts du développement web côté serveur, principalement avec le framework Django (Python). Les sujets abordés incluent :

- Création de sites web dynamiques
- Manipulation de bases de données (ORM Django)
- Gestion des utilisateurs et authentification
- Utilisation de Bootstrap pour le front-end
- Déploiement avec Docker
- Gestion de projet avec Git

## Structure du dépôt

- **TP1 à TP6** :
  - Mise en place de sites web statiques puis dynamiques
  - Scripts Bash pour la génération de pages HTML
  - Développement d'applications Django complètes (modèles, vues, templates, formulaires, authentification, etc.)
  - Utilisation de Bootstrap et de bibliothèques front-end
  - Déploiement via Docker
- **CT** :
  - Projet d'évaluation finale reprenant l'ensemble des compétences acquises

Chaque dossier contient un README détaillant les étapes, commandes et choix techniques propres au TP ou au CT concerné.
